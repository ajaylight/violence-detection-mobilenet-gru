# Real-Time Violence Detection and Alert System using MobileNet-GRU

## Overview

A deep learning-based surveillance system that detects violent activities from live video streams and triggers real-time alerts.

The system uses a MobileNet-GRU architecture to analyze video frames and classify violent behavior.

## Features

- Real-time violence detection
- Live camera monitoring
- Alert generation system
- Incident logging
- Snapshot capture
- MobileNet-GRU deep learning model
